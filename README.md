# Todo App.

Todo app is a web application for managing daily tasks.


## Demo

 [Live Demo](http://a-hosseini.ir/projects/todo/)

### Features


- Progress Chart
- Custom priority
- Custom Categories
- Search and Filter
- Edit Tasks
- Save information (LocalStorage)
- Built with Vanilla js
- Responsive
